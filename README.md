# Mixin TUI

This is a project that has a TUI terminal user interface for mixin bot to run different actions like transfer asset, read asset balance and more. It should be written in typescript, and use @mixin.dev/mixin-node-sdk as basement. 

There's a project called mixin-cli (https://github.com/fox-one/mixin-cli) written in go, this project should most functionality mixin-cli has. but in TUI.

## Usage

1. Copy `mixin-config.json.example` to `mixin-config.json` and fill in your Mixin bot credentials.
2. Run `bun install` to install dependencies.
3. Run `bun run dev` to start the TUI.
